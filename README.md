# Розробка веб-орієнтованої інформаційної системи для реалізації побутової техніки та електроніки

«Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра [ВСП «ППФК НТУ «ХПІ»](https://www.polytechnic.poltava.ua)).

Керівник роботи - **Ковальова Наталія Володимирівна**.

## Завдання до дипломної роботи

інформаційну систему для реалізації побутової техніки та електроніки. Слід забезпечити стандартну для систем такого роду функціональність: можливість авторизації, багаторівнева категорія товарів, фільтрація товару за технічними характеристиками, додавання товару у кошик, можливість здійснення онлайн-купівлі, тощо. Можливість додавати нові товари, завантажувати фото, змінювати ціни та опис. Розроблена система та використані технології мають бути детально описані в пояснювальній записці.
